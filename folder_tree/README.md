# folder

## 概要 (Overview)

フォルダの構成を出力しますが、treeコマンドとほぼ同じ

## 動作要件 (Requirements)

Python3

## 使い方 (How to Use)

1. コマンドプロンプト等で目的のフォルダに移動

2. folder.pyをPython3で起動

3. output.txtに出力されます。
